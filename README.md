# JavaScriptで作るテトリス

職業訓練校で学習したことの集大成として制作した、JavaScriptによるテトリスです。

## プロジェクトの概要
プログラミングの基礎（制御構文、配列、イベント処理）の実践に加え、AIを「開発パートナー」として活用し、複雑なロジックの実装に挑戦しました。

## こだわったポイント
### 1. Tスピン判定の実装
テトリスの特殊ルールである「Tスピン」の判定ロジックに最も注力しました。
- 公式仕様（SRS）を調査し、回転前後のブロック位置を配列で管理。
- 周囲の壁やブロックの状況を条件分岐で判定するロジックを自力で構築・デバッグしました。

### 2. AI（Gemini）との協調開発
- アルゴリズムの設計や実装方針の相談にAIを活用。
- バグが発生した際はまずAIに原因を相談しましたが、AIで解決できない場合は自分でコードを読み返したり、問題箇所を一つずつ考察したりして対処しました。
- AIが生成したコードはそのまま使うのではなく、コードの解説を求めて一行ずつ内容を理解することを徹底しました。
- 読みやすさを意識し、AIと協力してコードのリファクタリングも行いました。

### 3. 今後の展望・挑戦したいこと
- テトリス開発を通じて「既存のルールを形にする」基礎を学びました。次は、特定の課題を解決するために、UI/UXデザインからロジックまでをゼロから設計する開発に挑戦したいと考えています。
